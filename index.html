Thoughts on Progressive Enhancement

Progressive Enhancement is a methedology that web developers use to make websites avaliable to a whole bunch of different people. It focuses on content first and allows people using different devices and browsers to access the same site. Progressive enhancement is different from graceful degredation because it solves a greater number of problems. It is important to modern web developers because it allows them to show their website to the multitude of new browsers and devices that have been created without worrying about whether or not the site will show correctly.
